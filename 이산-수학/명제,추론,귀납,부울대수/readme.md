### 추론,연역법과 귀납법
#### [명제, 추론, 귀납]

추론: 이미 참으로 알고 있는 명제들로부터 ‘새로운 참인 명제’를 찾아내려는 과정

p1 → p2  :  전제 → 결론

올바른 추론의 규칙을 ‘논리’라고 함

추론의 방법: 연역법 & 귀납법

- 귀납법
    
    개별적인 사실을 말하는 명제들로부터 일반적인 결론을 도출하는 방법
